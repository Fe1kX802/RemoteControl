Запустите main.py, предварительно создав в рабочей директории password.txt и записав в него пароль в числовом формате (нужен при первом заходе на сайт)
На сайт можно зайти только если компьютер и клиент находятся в одной wifi сети
localhost:8000

Установка зависимостей: python -m pip install psutil fastapi pydantic uvicorn
